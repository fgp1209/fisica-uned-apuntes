# MathNote Live

Hoja negra rápida para escribir matemáticas en móvil con MathLive.

## Uso

- Toca una línea: usa el teclado nativo de Android.
- `kbd`: fuerza volver al teclado nativo.
- `Σ`: abre/cierra el panel matemático. También intenta abrir el teclado virtual de MathLive; si Android/Chrome lo bloquea, queda el panel propio como fallback.
- Barra superior: funciones frecuentes.
- Desplegable: funciones en orden alfabético, pensadas para MM1/MM2/cálculo/álgebra.
- `↵` o Enter: nueva línea matemática.
- `copy`: copia todo el LaTeX.
- `src`: muestra el LaTeX acumulado.
- `clear`: borra el guardado local.

## Atajos de texto

También funcionan comandos simples + espacio dentro de una línea:

```text
fraccion
integral
sumatorio
limite
matriz2
matriz3
determinante2
determinante3
casos
alineado
cr
cauchy
laplace
residuo
residuos
laurent
taylor
```

Además:

```text
a/ + espacio  -> fracción con a arriba
```
