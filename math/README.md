# MathNote Live

Editor matemático HTML para móvil/tablet.

## Uso

Abrir `index.html` o subir la carpeta a GitHub Pages.

## Teclado

Teclado virtual propio, siempre visible:

- `abc`: disposición tipo Gboard.
- `123`: números y símbolos.
- `fx`: funciones matemáticas ordenadas alfabéticamente.
- `αβγ`: letras griegas y símbolos habituales.

No usa el teclado nativo Android ni el teclado virtual de MathLive.

## Entrada rápida

- `_` para subíndice.
- `^` para superíndice.
- `a/` + `space` crea una fracción.
- `fraccion`, `integral`, `sumatorio`, `limite`, `matriz2`, `matriz3`, `determinante3`, `cauchy`, `laplace`, `laurent`, `taylor` + `space` insertan plantilla.

## Datos

Autosave en `localStorage`. Botón `copy` copia el LaTeX acumulado.
