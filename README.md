# Biología HTML — UNED Física

Proyecto estático para publicar guías de estudio en HTML responsive y exportables a PDF A4.

## Archivos iniciales

- `index.html`: índice general por temario.
- `biologia-t1.html`: Tema 1 — Introducción, teoría celular, evolución y taxonomía.
- `assets/styles.css`: estética común, responsive y print A4.
- `assets/app.js`: configuración MathJax e impresión.
- `templates/tema-template.html`: plantilla para próximos temas.
- `pdf/`: carpeta prevista para PDFs exportados manualmente.

## Uso local

Abrir `index.html` en el navegador.

## Publicación GitHub Pages

Subir el contenido del directorio a un repositorio y activar Pages desde la rama principal.

## Exportar PDF

Abrir un tema en navegador → botón `Imprimir / guardar PDF` → destino `Guardar como PDF`.
